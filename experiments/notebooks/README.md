# Experiments

Place Jupyter notebooks and one-off research scripts here.

## Naming convention

```
YYYY-MM-DD_short-description.ipynb
```

## Guidelines

- Keep notebooks reproducible  pin dependencies, set random seeds.
- - Store large artefacts (model weights, datasets) outside the repo and reference them by path or URL.
  - - Summarise key findings in a markdown cell at the top of each notebook.
